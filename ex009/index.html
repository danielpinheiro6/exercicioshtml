Trabalhando com listas

Listas ordendadas

Existe um comando chamado ordered list, cujo código é <ol>, que ordena uma série de coisas em listas, automaticamente, sem que seja necessário ficar adicionando o número de cada linha. Para isso, é necessário escrever a série de coisas dentro da tag <li>. Antigamente era necesário fechar a tag, mas atualmente não há mais essa necessidade. Funciona assim:

    <ol>
        <li>Acordar
        <li>Tomar banho
        <li>Tomar café
    </ol>

Esse código vai aparecer assim:

  1. Acordar
  2. Tomar banho
  3. Tomar café

Observação

Não precisa acidionar o </li>.

Personalizando listas

Podemos adicionar o parâmetro type dentro da tag Ordered List. Será possível escolher entre cinco tipos diferentes:

  1. Tipo 1 (numeral)
  1. Tipo 2 (letra maiúscula)
  1. Tipo 3 (letra minúscula)
  1. Tipo 4 (números romanos maiúscula)
  2. Teste
  1. Tipo 5 (números romanos minúsculo)
  2. Teste

É possível ainda informar em qual ordem deseja começar a lista, utilizando o parâmetro start:

  1. Tipo 1

Listas não-ordenadas

Se listas ordenadas são Ordered Lists, listas não-ordenadas são Unorderes Lists.
