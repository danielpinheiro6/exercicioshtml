Código-fonte

O comando document.getElementById('teste') é escrito em linguagem JavaScript. Note, todavia, que as letras do referido comando estão muito bem caracterizadas, diferente das demais. Isso se dá pelo uso do código <code>.

Preste atenção! O código <code> não organiza com identação. Fica tudo bagunçado.

num = int(input('Digite um número')) if num % 2 == 0: print(f'O número {num} é PAR) else: print(f'O número {num} é ÍMPAR) print('Fim do programa')

Olha agora utilizando o comando <pre> como fica:


        num = int(input('Digite um número'))
        if num % 2 == 0:
            print(f'O número {num} é PAR)
        else:
            print(f'O número {num} é ÍMPAR)
            print('Fim do programa')

And that's how you win the game!

Citações completas

Segundo os gregos, no seu livro Os 300 de Sparta:

Lorem ipsum, dolor sit amet consectetur adipisicing elit. Architecto ducimus deleniti impedit harum modi deserunt, quis debitis inventore iusto ab eveniet cumque soluta fugit nemo ratione sequi laudantium fugiat consequatur.

Presta atenção em outra coisa, é importante você deixar avisado para o navegador de onde você extraiu essa fonte. Por isso, dentro do código <blockquote> utilize o código cite, dessa forma:

        <blockquote cite="endereço da fonta na web>

Observação

Isso não fará aprecer nada de diferente no front end, mas o navegador e os serviços de busca saberão. Apenas pelo exemplo, vai aparecer assim:

Lorem ipsum, dolor sit amet consectetur adipisicing elit. Architecto ducimus deleniti impedit harum modi deserunt, quis debitis inventore iusto ab eveniet cumque soluta fugit nemo ratione sequi laudantium fugiat consequatur.

Texto invertido

A tag <bdo> virá com um complemento dir= para você escrever a sintaxe rtl, que significa right to left vai te pedir se você quer fazer o texto da direito para a esquerda ou ao contrário. É bizarro. O texto o brasil é o país das maravilhas seria escrito assim assim:

    <bdo dir="rtl">O Brasil é o pais das maravilhas.

Mas vai aparecer assim:

O Brasil é o país das maravilhas.
