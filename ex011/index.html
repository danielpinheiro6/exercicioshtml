Usando links externos

Abrir links em novas abas

Este exercício vai te mostrar como adicionar links ao seu código e a seu site. É simples. Por exemplo, caso queira acessar meu repositório no GitHub, clique aqui.

Por que isso acontece?

Porque adicionamos os atributos target="_blanc" e também o atributo rel="external". Isso também é interessante para o SEO. Com isso, o código fica assim:

    <a href="endereço do site" target="_blanc" rel="external">
